Utilizes Data sets from Kaggle.com on football events and player shooting attributes.

Focuses on using event data such as shot location, shot placement, player shooting abilities, time, home or away, body part used, and assist method to create a model that can predict the outcome of a shot.Logisitic Regression was mainly used, while a LASSO penalty was tested did not provide useful results.

Files are in a private reposititory as this was a project for a course.
